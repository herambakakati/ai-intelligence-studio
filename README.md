# 🚀 AI Intelligence Studio

A premium AI-powered dashboard built using Streamlit that combines **Anime Recommendation System** and **Social User Clustering** into one intelligent platform.

---

## 🌟 Features

### 🎬 Anime Recommendation System

* Content-based filtering using **TF-IDF**
* Similarity calculation using **Cosine Similarity**
* Displays top recommended anime with ratings
* Fetches posters dynamically using API

### 👥 Social User Clustering

* User segmentation using **KMeans Clustering**
* Dimensionality reduction using **PCA**
* Interactive visualization using **Plotly**
* AI-generated insights for each segment

### 🎨 UI & Experience

* Dark futuristic UI
* Glassmorphism design
* Responsive layout
* Interactive charts

## 📊 How It Works

### 🔹 Anime Recommendation

* Combines `genre + type`
* Converts text into vectors using TF-IDF
* Uses cosine similarity to recommend similar anime

### 🔹 Clustering Model

* Cleans and standardizes dataset
* Applies PCA for dimensionality reduction
* Uses KMeans to group users into segments
